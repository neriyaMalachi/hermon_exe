This project demonstrates the principles of **Object-Oriented Programming (OOP)** in JavaScript by creating a simple system for managing a library.  
It includes classes, inheritance, polymorphism, and clean file separation.

---

## Project Goals
- Understand how **Classes** work in JavaScript  
- Learn **constructor**, properties, and methods  
- Practice **Inheritance** (extends + super)  
- Understand **Polymorphism** through method overriding  
- Work with a clean folder structure using **ES Modules**
